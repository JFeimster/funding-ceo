# Moonshine Capital – Funding Agency Training Landing Page

Single-page landing site to generate leads for Moonshine Capital's free funding agency training.

## Files

- `index.html` – main landing page markup
- `styles.css` – layout, typography, and responsive styling

## How to Use

1. Put `index.html` and `styles.css` in the same folder.
2. Open `index.html` in your browser to preview locally.
3. To deploy on Vercel, Netlify, or GitHub Pages, upload this folder as a static site.

## Integration Notes

- Replace the `<form action="#">` in `index.html` with your email/CRM or funnel URL.
- Update the logo, favicon, and any tracking scripts (pixel, analytics) as needed.
